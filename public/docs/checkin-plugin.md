# 签到系统插件

## ✨ 主要功能

- 每日签到积分系统
- 连续签到奖励
- 积分排行榜
- 签到提醒功能
- 自定义签到奖励规则

## 🚀 快速开始

1. 下载插件文件到 `plugins/checkin`
2. 首次运行会自动创建数据库
3. 重启 Astrabot
4. 使用 `/签到` 开始签到

## ⚙️ 配置说明

自定义签到奖励：

```json
{
  "daily_points": 10,
  "continuous_bonus": 5,
  "max_continuous": 30
}
```

## 📝 使用示例

- `/签到` - 每日签到
- `/签到信息` - 查看签到记录
- `/排行榜` - 查看积分排名
